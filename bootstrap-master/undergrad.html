Network Intrusion Detection System.

Undergrad project

Introduction


Network Intrusion Detection System or NIDS is a software tool used to monitor network for failure or irregularities automatically.The tool is also aimed in assisting the Network Administrator in managing and monitoring the network .NIDS is required to detect all types of malicious network traffic and computer usage.

NIDS is an independent platform, which identifies intrusions by examining network traffic and monitors multiple hosts. NIDS gain network traffic by connecting to a hub, network switch.


System design


Database design is recognized as standard of MIS and is available for every computer system virtually. while designing a database, several objects must be considered. They include the redundancy of easy to leaving and use, data dependency, more information at low cost, accuracy and integrity. Various tables used in our applications are login, intruder, modal, detection.
