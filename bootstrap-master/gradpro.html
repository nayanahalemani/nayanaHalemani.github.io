Undergrad project

Introduction


This project is designed for software engineering development course. The main objective of this project was focusing and implementing it in accordance with waterfall model.

The game will consist of 3 levels easy, medium, and hard. These difficulty levels will have a pre-determined amount of questions that will be randomized every time the game is played. This type of gaming structure will ensure that if the user plays the game three times in a row that they will not get the same questions three times in a row. Each level’s questions will be worth a certain amount of points. The user has to accumulate a certain number of points before they can advance to the next round. Unbeknownst to them that one of the question will be a hidden bonus that could potentially advance them to the next round without going through the process of answering every question in the round. What makes the game such a great learning tool involves potential incorrect answers, if the user gets a question wrong the correct answer will be shown to them educating them while also telling them to start over. The questions will cover a variety of topics from history, science, geography, and entertainment. The user’s total score will be calculated at the end of the game. System design


Goal of the project:The Smart Shot Quiz Software is to present a user-oriented multiple-choice quiz program with different topics and different levels

Objectives

  1. Topics: 4 different topics are available, including history, science, art and literature.
  2. Difficulty Levels: for each topic, there are questions at different levels: easy, medium and hard.
  3. Bonus: questions will appear randomly, users will get more points if they answer the bonus questions correctly.
  4. Quiz modes: Exam mode with time limits, users are required to submit the answer before time- and Casual mode, the time limits are removed.
  5. Privacy and easy access: the software allows the user to create an account so his/her quiz results won’t be shared with other users. For users who want to start a quick quiz without logging on or registration, a quick start quiz feature is also available
  6. Status review: during the quiz, the user can check the quiz status, including the answered questions, correct answers and accumulated scores.
  7. Progress storage: the software has a progress saving functionality; user can save the current quiz and continue later on.
  8. User feedback.
